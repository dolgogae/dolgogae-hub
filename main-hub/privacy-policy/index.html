개인정보처리방침

시행일: 2026년 4월 30일  |  최종 수정일: 2026년 4월 30일

1. 개요

본 서비스는 사용자의 개인정보를 소중히 여기며, 「개인정보 보호법」 및 관련 법령을 준수합니다. 본 방침은 본 서비스가 수집하는 정보, 사용 방식, 보관 기간, 사용자의 권리에 대해 설명합니다.

2. 수집하는 개인정보

2.1 회원가입 및 인증 정보

  • Google 로그인: 이름, 이메일 주소, 프로필 사진 URL (Google OAuth 제공)
  • Apple 로그인: 이름, 이메일 주소 (Apple Sign-In 제공)
  • 이메일/비밀번호: 이메일 주소, 암호화된 비밀번호

2.2 서비스 사용 데이터

  • 사용자가 저장한 URL 및 해당 페이지 메타데이터 (제목, 설명, 썸네일)
  • 사용자가 생성한 카테고리 및 폴더 구조
  • 월별 링크 저장 횟수 (요금제 관리 목적)

2.3 결제 정보

  • Google Play를 통한 인앱 구매 처리 시 구매 토큰 (Google이 직접 처리하며 본 서비스는 검증 목적으로만 사용)
  • 본 서비스는 신용카드 번호 등 결제 수단 정보를 직접 수집하지 않습니다.

2.4 자동 수집 정보

  • Firebase 익명 분석 데이터 (사용 패턴, 오류 로그)
  • 기기 식별자 (광고 식별자 포함, 광고 제공 목적)

3. 개인정보 이용 목적

목적 수집 항목
회원 식별 및 인증 이름, 이메일, 프로필 사진
링크 저장 및 AI 자동 분류 서비스 제공 저장 URL, 메타데이터
요금제 관리 (무료/프리미엄) 월별 사용량, 구매 토큰
광고 제공 (무료 이용자) 기기 광고 식별자
서비스 개선 및 오류 분석 익명 사용 로그

4. 제3자 제공 및 위탁

본 서비스는 다음 제3자 서비스를 사용하며, 해당 서비스의 개인정보처리방침이 적용됩니다.

서비스 제공자 목적 관련 방침
Firebase Auth / Firestore Google LLC 인증, 데이터 저장 Firebase 개인정보처리방침
Google Sign-In Google LLC 소셜 로그인 Google 개인정보처리방침
Apple Sign-In Apple Inc. 소셜 로그인 Apple 개인정보처리방침
OpenAI API OpenAI, LLC AI 링크 자동 분류 OpenAI 개인정보처리방침
Google AdMob Google LLC 광고 제공 (무료 플랜) Google 광고 정책
Google Play Billing Google LLC 인앱 결제 처리 Google Play 개인정보처리방침

본 서비스는 위 목적 이외에 사용자 데이터를 제3자에게 판매하거나 공유하지 않습니다.

5. 개인정보 보관 기간

  • 계정 데이터:회원 탈퇴 시 즉시 삭제
  • 저장 링크 및 카테고리:사용자가 직접 삭제하거나 탈퇴 시 삭제
  • 결제 기록:관련 법령에 따라 최대 5년 보관 후 삭제
  • 광고 식별자:Google AdMob 정책에 따라 관리

6. 사용자의 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다.

  • 조회:저장된 개인정보 확인
  • 수정:프로필 정보 변경 (앱 내 설정에서 가능)
  • 삭제:링크, 카테고리, 계정 삭제 (앱 내 설정 → 계정 삭제)
  • 처리 정지:광고 식별자 추적 중지 (기기 설정 → 광고 → 광고 ID 재설정/삭제)

계정 삭제 요청은 앱 내 설정 화면 또는 아래 이메일로 문의하실 수 있습니다.

7. 아동 개인정보 보호

본 서비스는 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 개인정보 보안

  • 모든 데이터는 Firebase를 통해 전송 및 저장 시 암호화됩니다.
  • 인증 정보는 Firebase Authentication을 통해 안전하게 관리됩니다.
  • OpenAI API 호출 시 URL 메타데이터만 전송되며, 사용자 식별 정보는 포함되지 않습니다.

9. 개인정보처리방침 변경

본 방침이 변경될 경우 앱 내 공지 또는 이메일을 통해 사전에 안내합니다. 변경 후에도 본 서비스를 계속 사용하면 변경된 방침에 동의한 것으로 간주합니다.
